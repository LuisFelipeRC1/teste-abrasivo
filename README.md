Boa noite,
Se você está recebendo este e-mail, é porque foi selecionado para a segunda e última etapa do processo seletivo para a vaga de estágio em desenvolvimento web na Abrasivo Digital.
Para esta fase, disponibilizamos um arquivo no Figma contendo o layout completo da landing page, com textos, imagens, backgrounds e demais elementos visuais. O seu desafio será transformar esse protótipo em uma página funcional utilizando HTML, CSS e JavaScript.
Para os vídeos da página, você pode utilizar os seguintes links:
Vídeo da sessão inicial: https://www.youtube.com/watch?v=BOLSMaepKU4
Vídeo final: https://www.youtube.com/watch?v=VZrQs3gulyI
Objetivo do teste
Desenvolver uma landing page fiel ao layout proposto no Figma, garantindo qualidade visual, estrutura e funcionalidade.
Critérios de avaliação
Fidelidade ao layout original do Figma
Precisão em cores, tipografia e dimensões
Organização e espaçamento dos elementos
Responsividade para diferentes dispositivos como smartphone, tablet e desktop
Interatividade e criatividade na aplicação de animações, carrosséis e recursos visuais
Navegação funcional no menu, com direcionamento para as seções da página
Implementação do carrossel automático na seção de marcas
Validação de formulário, incluindo máscara para telefone
Boas práticas de programação e codificação
Observações importantes
Os links de Login e Suporte não precisam ser funcionais, mas devem estar presentes
O formulário não precisa realizar envio de dados, apenas validação
O captcha da Cloudflare pode ser desconsiderado
Se desejar, você pode incrementar o formulário com funcionalidades como preenchimento automático via CEP ou seleção dinâmica de cidades por estado
